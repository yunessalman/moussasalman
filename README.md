# Moussa Salman — Artist Website

Static site. No build step.

## Structure
- `index.html` — Home
- `limitededitions.html` — Limited Editions (pricing + per-edition inquiry)
- `biography.html`, `reviews.html`, `exhibitions.html`, `awards.html`, `contact.html`
- `site.js` — artwork catalogue + lightbox + gallery + contact prefill
- `styles.css` — all styles
- `art/` — artwork images (01–37.jpg) + portrait

## Deploy (GitHub Pages)
1. Create a new repo and push these files to the root.
2. Settings → Pages → Branch: `main`, folder `/ (root)`.
3. Your site goes live at `https://<user>.github.io/<repo>/`.

## Local preview
Open `index.html` in a browser, or run `python3 -m http.server`.
